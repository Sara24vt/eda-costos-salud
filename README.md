Análisis Exploratorio de Datos en Salud

Descripción
Este proyecto presenta un análisis exploratorio de datos (EDA) en el sector salud, utilizando un dataset público y anonimizado, con el objetivo de identificar patrones, relaciones entre variables y posibles insights relevantes para la toma de decisiones.

Datos

Fuente: Dataset público de salud (anonimizado)
Tipo: Datos estructurados
Variables principales: edad, sexo, IMC, hábito de fumar, región y costos médicos

Proceso de análisis

Importación y exploración inicial de los datos
Revisión de estructura y tipos de variables
Validación de valores faltantes y atípicos
Visualización de datos
Generación de insights

Herramientas utilizadas

Python
Jupyter Notebook
Pandas
Matplotlib
Seaborn

Principales insights

Los costos médicos tienden a aumentar con la edad, con mayor variabilidad en edades avanzadas.
El tabaquismo se identifica como uno de los factores con mayor impacto en el incremento de los costos médicos.
Existen diferencias regionales en los costos médicos.
